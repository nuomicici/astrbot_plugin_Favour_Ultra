# 更新须知
更新后如果无法正常加载，请依次尝试：
1. 点击插件重载按钮
2. 重启框架
3. 重新安装该插件
4. **必看！**：如果后台显示“无法识别好感度标签”，请先尝试在预设对话中添加如下内容并清空历史对话记录！（若无效请前往反馈）

## 更新日志
### v3.1.3 (2026-03-13)
*   **prompt更新**：使用更规范的XML格式注入，并要求达到上限后不再提高好感度防止输出出现错误。
*   如遇问题，可以复制下方链接，于 手动下载 -> 链接下载 处手动回退
```
https://github.com/nuomicici/astrbot_plugin_Favour_Ultra/tree/backuup/
```
### v3.1.2（2026-03-10）
*   **命令补全**：添加多个管理命令，方便管理员全局管理数据
*   **冷暴力焕新**：将判定方式修改为好感度连续降低触发
*   **输出优化**：添加分页功能，防止t2i输出过长图片导致无法正常发送
*   **显示优化**：查看单人好感度直接文本输出，减少t2i负载

### v3.1.0 (2026-01-09)
*   **架构升级**：弃用 JSON 文件存储，全面迁移至 SQLite 数据库。
*   **配置重构**：引入嵌套配置结构，新增 `enable_cold_violence` 开关。
*   **功能增强**：
    *   新增 `跨会话修改` 命令，支持远程运维。
    *   优化 `查询全部好感度` 逻辑，支持按会话分组并智能隐藏私聊数据。
    *   多人查询命令全面适配 T2I 图片输出，表格化展示更清晰。
*   **修复**：修复了部分全局命令鉴权失败的问题。

### v3.0.2 (2025-12-27)
*   优化冷暴力 Prompt 注入逻辑，防止 LLM 误学习上下文。

### v3.0.0 (2025-12-05)
*   新增关系唯一性判断（如：只能有一个“恋人”）。
*   优化 T2I 输出容错率。
